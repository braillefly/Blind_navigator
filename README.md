```
pip install openai PyQt5 opencv-python
```

## Run 

Run with a video 

```
python LLAMA_3_2_navigation.py --video <video path> --api <api key>
```
Run with a camera

```
python LLAMA_3_2_navigation.py --video <int correpsonding to the camera> --api <api key>
```
